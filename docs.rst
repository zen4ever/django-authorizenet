Usage
=====

Installation
------------

Install from `PyPI`_:

.. code-block:: bash

    $ pip install django-authorizenet

.. _PyPI: https://pypi.python.org/pypi/django-authorizenet/


Quickstart
----------

Add ``authorizenet`` to ``INSTALLED_APPS`` in your settings file:

.. code-block:: python

    INSTALLED_APPS = (
        ...
        'authorizenet',
    )
